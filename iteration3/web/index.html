FlashPhoto

This is the website for 11amGroup02. We create an application that can edit the JPEG or PNG image. The image in the right hand is the main tool window will show up when you run the application. There are different tools and filters you can use to edit and save the images or you can create your own piece. Check the course website CSCI 3081W. for more details.

During the process, we use several libraries to help us, including GLUI, JPEG, PNG and a library create by us called libphoto. You can find a link to documentation for all the code that you have auto-generated using doxygen under the developer page.
