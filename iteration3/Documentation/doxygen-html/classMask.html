Detailed Description

All tools need to declare their mask to identify their shapes.

.h PhotoShop Proj

Created by Seth Johnson on 2/9/15.. Copyright (c) 2015 Seth Johnson. All rights reserved.Mask is a wrapper for the array of floats representing the shape of a tool. It is pure virtual, because a mask does not inherently have a shape. Shapes are defined implicitly with getIntensity().
