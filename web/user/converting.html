Converting and Segmenting a CT Scan Using Mia

The feature that sets Mia apart from other image-editing software is its ability to apply adjustments to an entire stack of images easily, such as a stack of CT scan images with a single command in command line mode. It may be helpful to play around with graphics mode to decide your desired settings before filtering and saving hundreds of images in your system.

Once you know your desired settings, however, you will find command line mode to be extremely quick and easy to use. In this tutorial, I will walk through the process of converting and editing a stack of CT images in command line mode.

To process a stack of numbered image files with the command line, use the following command:

./Mia ct_scan_result###.png -blur 5 -quantize 4 ct_scan_processed###.jpg

The first argument, ct_scan_result###.png, is the name of the input files, but with the identifying number replaced with a###.png

This set of files has a 3-digit identifying number, but if, for example, your batch of files has a 5-digit identifying number, use five #'s instead. The next arguments are the filters we want to apply to the images. -blur represents the Blur filter, and itâs followed by the amount of blur we want.

Applying a blur filter reduces the noise in the image. -quantize represents the Quantize filter, and we specified 4 for 4 segments, which will show the different regions of tissue more clearly. There are seven different filters in total that can be applied to each image using the following arguments. If the filter requires an input, like Blur and Quantize do, the input number must follow the filter name:

Sharpen filter:
-sharpen

Edge Detect filter:
-edgedetect

Threshold filter:
-thresh

Quantize filter:
-quantize

Blur filter:
-blur

Saturate filter:
-saturate

Multiply RGB filter:
-multrgb
[note: this filter requires 3 inputs, for the red value, the blue value, and the green value respectively]

The last argument is the output file name, again with the #âs at the end representing the identifying number. Notice that we can convert to .png to .jpg file extension, if we want. After running this command, there should be a new filtered image saved as the output file name for each of the numbered files that match the pattern.

Hours of work, done in two seconds!
